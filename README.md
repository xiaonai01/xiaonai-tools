# 小奈工具集 (XiaoNai Tools)

这是一个实用的工具集合，包含各种自动化脚本和实用工具。

## 项目结构

```
xiaonai-tools/
├── README.md
├── requirements.txt
├── scripts/
│   ├── file_organizer.py      # 文件整理工具
│   ├── text_processor.py      # 文本处理工具
│   └── web_scraper.py         # 网页抓取工具
└── utils/
    ├── file_utils.py          # 文件工具函数
    └── text_utils.py          # 文本工具函数
```

## 使用方法

1. 安装依赖：`pip install -r requirements.txt`
2. 运行脚本：`python scripts/<script_name>.py`

## 功能特点

- 文件整理：自动按类型整理文件
- 文本处理：批量处理文本文件
- 网页抓取：简单易用的网页数据抓取
